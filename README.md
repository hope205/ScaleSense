# ScaleSense

An AI agent that assits human resource managers to rank candidates that are most suitable for a particular job role



## 🎯 Project Objective
Human resource manager are usually faced with the task of going through the applications of several canddates which usually time consuming. **ScaleSense** addresses this by building an AI agent that can analyse each of the resumes, extract the revelant metadata and identify the most relevant candidates for the role
